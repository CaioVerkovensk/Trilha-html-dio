Servidores

Conceitos Servidor:é um computador equipado com um ou mais processadores, bancos de memória, portas de comunicação, softwares e, ocasionalmente, algum sistema para armazenamento de dados, como hard disks internos ou memórias SSD.

  • Pode ser local ou online.
  • Tende a ser uma máquina mais robusta, para servi a muitos acessos
  • Tipos de servidor (Arquivos, Segurança, Streaming, E-mail, Web)
  • Pode ser uma máquina comum ou um Data Center (conjunto gigante de servidores)

    • Tipos de Servidores

  • Servidor Proxy: Um servidor proxy proporciona um gateway entre os usuários e a internet. É um servidor chamado de “intermediário”, porque fica entre os usuários finais e as páginas da web que eles visitam on-line.
  • Servidor Firewall: Servidor voltado a acesso externo
  • Servidor Web server: "Servidor web (web server)" pode referir ao hardware ou ao software, ou ambos trabalhando juntos.
  • Servidor e-mail: voltado a e-mails
  • Servidor data base web:
  • DNS : Domain Name Service ele redireciona o nome para um IP especifico

    • Software (Parte Logica)

    Referente ao software, um servidor web inclui diversos componentes que controlam como os usuários acessam os arquivos hospedados (armazenados para disponibilização), no mínimo um servidor HTTP. Um servidor HTTP é um software que compreende URLs (endereços web) e HTTP (o protocolo que seu navegador utiliza para visualizar páginas web.

    Hardware (parte Física)

    Referente ao hardware, um servidor web é um computador que armazena arquivos que compõem os sites (por exemplo, documentos HTML, imagens, folhas de estilo, e arquivos JavaScript) e os entrega para o dispositivo do usuário final. Está conectado a Internet e pode ser acessado através do seu nome de domínio (DNS), como por exemplo mozilla.org.

    Requisição HTTP (Quando o navegador pede alguma informação)

    Quando chega uma requisição no servidor ele ativa o servidor web e ele envia as informações em Html, css, Javascript, imagens e texto.

    Hospedagem

    Colocar os arquivos do site em um servidor que fica disponível na internet que ficam disponível para os clientes acessarem O provedor de hospedagem fornece os dados de acesso FTP (File Transfer Protocol) ao servidor. Depois de hospedar registrar o nome do site o DNS no registro.br
